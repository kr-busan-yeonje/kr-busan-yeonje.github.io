# 연제구 · 부산광역시

> 연제구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-yeonje.github.io/**

연제구는 부산광역시청과 부산고등법원·부산지방법원·부산지방검찰청이 한데 모인 부산의 행정·사법 중심지입니다. 구 이름은 연산동(蓮山洞)의 '연(蓮)'과 거제동(巨堤洞)의 '제(堤)'를 한 글자씩 딴 것으로, 두 법정동의 머리글자를 합쳐 지은 신생구의 지명입니다. 1998년 부산시청이 중구에서 연산동으로 옮겨오면서 관공서와 법조타운이 집적된 광역행정 거점으로 성격이 굳어졌습니다. 도심 한복판을 온천천이 흐르고 배산과 황령산 자락이 남쪽을 감싸, 관청가의 분주함과 하천 산책로의 여유가 공존합니다. 삼국시대 거대 봉분이 남은 연산동 고분군과 배산성지가 있어 부산에서 손꼽히는 고대 유적 밀집지이기도 합니다. 아시안게임과 월드컵을 치른 부산아시아드주경기장이 거제동에 자리해 체육·문화 행사의 무대 역할도 겸합니다.
